# 🧠 Storage Server Architecture  

---

## 📌 Overview  

This system uses ESP32 as a lightweight server to manage file storage and retrieval.

---

## 🔄 Workflow  

1. User connects via browser  
2. Sends file request  
3. ESP32 processes request  
4. File stored/retrieved from storage  
5. Response sent back  

---

## 🧩 Components  

- ESP32 Web Server  
- Client Browser  
- Storage Device  

---

## ⚡ Considerations  

- Data transfer speed  
- File system limitations  
- Memory usage
